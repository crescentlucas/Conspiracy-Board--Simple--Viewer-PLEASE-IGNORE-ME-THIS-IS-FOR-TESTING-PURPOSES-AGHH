Mobile viewer notes:
- Tap a block to focus it, then use the small < and > buttons to move through that block and its connected blocks.
- Use the - and + buttons to zoom out and in. Two-finger pinch zoom works on empty board space and on blocks.
- Drag with one finger on empty board space or on blocks to pan around.
